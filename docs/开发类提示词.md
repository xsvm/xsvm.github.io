# 开发类提示词
## 原型图模板
```
 `你是一位资深设计师，你非常了解 【Dribbble.com】(http://Dribbble.com) 以及 iOS/Google 等设计风格，拥有丰富的全栈开发经验及极高的审美造诣，擅长现代化设计风格，尤其擅长移动端设计及开发。`

`## 我想做的 App 需求是：`

我想做一个高情商回复生成器 App，名字叫会聊鸭。有 3 个页面：第一个页面是首页，显示一个 logo 加名字，然后有 4 个预设的模式：哄哄 TA/怒怼杠精/糊弄 TA/收到领导，还有一个自定义回复模式，这个页面点击任何一个模式都进入第二个页面：有一个大的输入框，输入框右下角可以选择上传聊天记录截图，上传完后自动 OCR 成聊天记录放到文本框里面，还有一个身份选择器（比如老公老婆，男女朋友，领导，杠精，通过预设模式进来会默认选中身份），下面有一个按钮”解读并生成回复“，点击后生成解读 + 回复。回复上面有复制按钮，还有一个”匿名发布“的按钮。第三个页面是广场，所有人都可以看到别人发布的高情商案例。

`## 我的要求：`

- `*页面元素尽量美观高级，遵守移动端设计规范，注重 UI 细节*`
- `*所有数据都用假数据，所有页面都可以点击交互*`
- `*图标使用 Lucide Static CDN 方式引入*`
- `*把设计生成在 design 目录下，每个子页面都用一个单独的 html, 为了方便在一个页面展示全，index.html 里面把所有的子页面都展示出来，用 iframe 来加载子页面*`
- `界面尺寸应模拟 iPhone 15 Pro，并让界面圆角化，使其更像真实的手机界面，顶部添加状态栏`
- `*思考过程仅思考功能需求、设计整体风格等，不要在思考时就写代码，仅在最终结果中输出代码*`

`请按照以上要求生成完整的高保真原型图（html）`
```
