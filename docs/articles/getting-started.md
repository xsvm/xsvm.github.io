# 开始使用

欢迎来到我的博客！这是一个使用 VitePress 构建的静态网站。VitePress 是一个基于 Vue 的静态网站生成器，它简单易用，性能出色。

## 为什么选择 VitePress？

- **简单**：VitePress 使用 Markdown 语法，让写作变得轻松自然
- **快速**：基于 Vite 构建，开发体验极致顺滑
- **优雅**：默认主题美观大方，同时支持自定义主题

## 快速上手

1. 创建新文章：在 `docs/articles` 目录下创建新的 Markdown 文件
2. 配置导航：在 `docs/.vitepress/config.ts` 中添加导航链接
3. 本地预览：运行 `npm run dev` 启动开发服务器
4. 构建部署：使用 `npm run build` 生成静态文件
